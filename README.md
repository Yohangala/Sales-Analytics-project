# Sales Performance & Customer Insights Dashboard

## Project Objective
Analyze global retail sales data to identify revenue trends, product performance, customer behavior, and regional insights using Python, SQL, and Power BI.

## Dataset
Global Superstore Dataset (Kaggle)

## Tools
- Python (Pandas, NumPy)
- SQL
- Power BI
